CSS specificity

CSS selector specificity is determined using the four numbers (a, b, c, d) in this order:

  • ID selector (a)
  • Class, attribute, and pseudo-class selectors (b)
  • Type and pseudo-element selectors (c)
  • Inline styles (d)
When comparing two selectors to determine which one has higher specificity, we start with the first number (a). If they are equal, we move on to the second number (b), and so on. The selector with the highest number at any point in the comparison has higher specificity.

Selector Specificity
p 0,0,1,0
.class 0,0,1,0
#id 0,1,0,0
div p 0,0,2,0
div > p 0,0,2,0
p + p 0,0,2,0
p ~ p 0,0,2,0
p[title] 0,1,1,0
p:hover 0,0,1,1
p::first-line 0,0,1,1
